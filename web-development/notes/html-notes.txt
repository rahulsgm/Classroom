HTML- hypertext markup language

Tables:
Tables allow web developers to arrange data into rows and columns.
Tag: //start tag: <table> //end tag: </table>
Each table row is defined with a <tr> tag. Each table header is defined with a <th> tag. Eache table data/cell is defined with a <td> tag.
By default, the text in <th> elements are bold and centered.
By default, the text in <td> elements are regular and left-aligned.
